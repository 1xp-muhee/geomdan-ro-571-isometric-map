# Geomdan-ro 571 Isometric 3D Map Prompt

홍보용 깔끔 버전의 아이소메트릭 3D 약도 제작용 자료입니다.

## 대상 위치
- **인천 서구 검단로 571**

## 목적
- 홍보물, 전단지, 브로슈어, SNS 업로드용으로 사용할 수 있는
  **깔끔하고 고급스러운 아이소메트릭 3D 지도 이미지** 제작

## Final Prompt

```text
Create a clean and polished promotional isometric 3D map illustration centered on 571 Geomdan-ro, Seo-gu, Incheon, South Korea. The destination should be the clear visual focus, shown as a refined highlighted building with an elegant location pin, soft glow, and subtle emphasis. Include surrounding roads and nearby building blocks in a simplified but realistic miniature city style. Keep the composition neat, minimal, and premium, suitable for a brochure, advertisement, or promotional handout. Use bright and professional colors, soft shadows, crisp edges, and a visually pleasing layout that is easy to understand at a glance. Add tasteful Korean text labeling the destination as “인천 서구 검단로 571”. Avoid clutter, excessive icons, cartoon exaggeration, or overly game-like rendering. The overall mood should feel modern, trustworthy, upscale, and easy to read.
```

## Negative Prompt

```text
messy composition, overly complex roads, distorted buildings, unrealistic perspective, dark colors, cluttered labels, low detail, low resolution, cartoonish style, childish look, exaggerated 3D game map, too many markers, confusing layout, bad typography, warped text
```


## Format Hints
- **정사각형 SNS용**: centered composition, balanced layout, optimized for square format
- **가로 배너용**: wide horizontal composition, destination slightly off-center with roads flowing naturally across the frame
- **인쇄 전단용**: clean white background, brochure-ready composition, premium print-friendly colors

## Note
현재 저장소에는 **이미지 생성 프롬프트와 제작 가이드**가 포함되어 있습니다.
실제 이미지 렌더링은 이미지 생성 도구(Nano Banana Pro, Midjourney, DALL·E, Gemini Image 등)에 위 프롬프트를 넣어 진행하면 됩니다.
